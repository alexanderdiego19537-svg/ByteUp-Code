# 📊 modelos/ (models/) — Las "plantillas" de datos de la app

Los modelos definen la estructura de los datos que usa la app:

| Archivo | Qué hace |
|---|---|
| `user_model.dart` | **Usuario** — Guarda nombre, email, XP, racha, corazones, insignias y progreso por lenguaje |
| `lesson_model.dart` | **Lección** — Define la pregunta, opciones de respuesta, respuesta correcta y tipo (código/texto) |
