| Archivo | Qué hace |
|---|---|
| `home_screen.dart` | Pantalla de INICIO (dashboard principal) |
| `learn_screen.dart` | Pantalla de APRENDER (mapa de lecciones) |
| `lesson_screen.dart` | Pantalla de LECCIÓN (preguntas interactivas) |
| `ranking_screen.dart` | Pantalla de RANKING (tabla de posiciones) |
| `profile_screen.dart` | Pantalla de PERFIL (estadísticas del usuario) |
| `auth_screen.dart` | Pantalla de LOGIN/REGISTRO |
| `splash_screen.dart` | Pantalla de CARGA (la primera que aparece al abrir la app) |
