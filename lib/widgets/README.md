# 🧩 componentes/ (widgets/) — Piezas reutilizables de la interfaz

Los widgets son como "bloques de LEGO" que se usan en múltiples pantallas:

| Archivo | Qué hace |
|---|---|
| `fon_master_widget.dart` | El personaje Fon Master (dibujado con código, no imágenes) |
| `draggable_fon_master.dart` | Fon Master que puedes mover con el dedo — 1 tap=mover, 2 taps=chat |
| `background_particles.dart` | Los puntitos verdes flotantes del fondo de las pantallas |
| `typewriter_text.dart` | Texto que aparece letra por letra (efecto máquina de escribir) |
| `hover_scale.dart` | Hace que una tarjeta se "hunda" al tocarla |
| `rank_card.dart` | Tarjeta que muestra el rango con emoji y barra de progreso |
| `lesson_card.dart` | Nodo del mapa de lecciones (bloqueada / activa / completada) |
